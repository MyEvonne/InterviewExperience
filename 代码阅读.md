# 代码阅读
1. **阅读，求输出**
```objective-c
TestObject *object1 = [[TestObject alloc] init];
__block TestObject *object2 = [[TestObject alloc] init];
object1.name = @"Mike";
object2.name = @"Sean";
__block int vi = 1;
void (^handler)(NSString *) = ^(NSString *name) {
  object1.name = name;
  object2.name = name;
  vi = 2;
}
handler(@"Lucy"); 13
NSLog(object1.name);
NSLog(object2.name);
NSLog(@"%i", vi);
```
输出：
```
Mike
Lucy
2
```
